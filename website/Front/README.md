# frontend

React SPA (Vite + Tailwind). Lets operators view connected devices, stream screens, control mouse & keyboard, run shell commands, and build custom client executables.

## Features

- Dashboard with live connection list
- Real-time screen streaming with latency display
- Mouse & keyboard remote control
- Multi-monitor switching
- Shell terminal
- Client builder with live progress
- User settings & account management

## Dev Notes

```bash
npm install
npm run dev
```

Edit `src/config.json` to point `BASE_API` at your backend instance.
